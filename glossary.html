Glossary

HTML

    1. <a href=>

    Creates hyperlinks to other pages, files, email addresses, or page sections.

    Key Points

  1. href specifies the destination URL (absolute or relative).
  2. You can open in a new tab with target="_blank" plus rel="noopener noreferrer".
  3. Can link to specific page sections with #id.
  4. Supports protocols like mailto: and tel: for email or phone links.

    2. <table>

    Displays tabular data in rows and columns.

    Contains one or more <tr> (table rows).

  1. <td> represents standard data cells; <th> is a header cell.
  2. Optional grouping tags: <thead>, <tbody>, <tfoot>.
  3. Use CSS (border-collapse, padding, etc.) for styling—avoid for general page layout.

    3. <img>

    Embeds images into a web page.

    <src> specifies the image file path (local or remote).

  1. <alt> provides alternative text for screen readers and SEO.
  2. Width/height can be set with HTML attributes or CSS.
  3. Combine with max-width: 100% for responsiveness.

    4. <ol> + <li>

    Ordered list where each <li> is a numbered item.

    Numbers items automatically; default is decimal (1,2,3).

  1. <type> attribute changes numbering style (A, a, i, I).
  2. <start> attribute sets the first number.
  3. Lists can be nested inside others.

CSS

    1. background-color

    Sets the background color of an element.

    Key Points

  1. Accepts color names, hex (#ff0000), rgb(), or hsl().
  2. Inherits only if explicitly set on a parent with inherit.
  3. Can be combined with background-image or gradients.

    2. margin and padding

    Control space outside (margin) and inside (padding) an element`s border.

    Key Points

  1. Shorthand syntax: margin: top right bottom left;.
  2. auto value (e.g., margin: 0 auto;) centers block elements horizontally.
  3. Can use units like px, em, %, or rem.

    3. border

    Adds a line around an element.

    Key Points

  1. Shorthand: border: width style color; (e.g., 1px solid #333).
  2. Can target specific sides (border-top, border-left).
  3. Works well with border-radius for rounded corners.
